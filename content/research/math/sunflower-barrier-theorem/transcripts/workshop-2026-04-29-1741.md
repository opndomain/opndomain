## Problem

PROVE THE FOLLOWING THEOREM (or disprove it with a counterexample):

There exist absolute constants c > 0 and w_0 such that for every w >= w_0, there exists a w-uniform family H_w with |H_w| >= (c log w)^w satisfying:

(1) H_w is kappa-regular with kappa = Theta(log w): for every T, Pr[T subseteq A] <= kappa^{-|T|}.

(2) No local-link regularization procedure can reduce H_w to C-spread (for any constant C) without losing a factor of exp(Omega(w log w)) in family size.

Here a 'local-link regularization procedure' is any iterative algorithm that:
- Maintains a current family F_t
- At each step, selects a single offending witness T_t with |F_t cap {A : T_t subseteq A}|/|F_t| > C^{-|T_t|}
- Replaces F_t by a subfamily of F_t or by the link (F_t)_{T_t}
- Terminates when no offending witness remains

The theorem says this class of algorithms — which includes the core regularization step in ALWZ (Lemma 2.6 + 2.8), Rao (post-Lemma 2), and BCW — MUST lose a logarithmic factor. The log in (C log w)^w is not a proof artifact; it is intrinsic to the method.

ACCEPTABLE OUTCOMES:
(a) Construct an explicit barrier family H_w and prove (1) and (2)
(b) Prove (2) for an abstract adversary without explicit construction
(c) Prove a weaker version: barrier on Theta(log w) discrete scales (not all scales)
(d) Disprove: exhibit a local-link regularizer that achieves C-spread with only exp(O(w)) loss on ALL kappa-regular families

PRIOR RESULTS FROM OUR RESEARCH:
[P1] The log enters at the spread-to-satisfying step, not at core extraction
[P2] Block-transversal T_{m,w} has exact p-biased hitting probability (1-(1-p)^m)^w, forcing kappa = Omega(log w) for independent sampling
[P3] Any architecture going through 'large family -> kappa-regular -> independent p-biased hitting -> sunflower' cannot beat (C log w)^w
[P4] Links of T_{m,w} are self-similar: bounded link recursion preserves the same obstruction
[P5] The ternary-block cap-set model (sunflower = 3-AP in F_3^w) is the correct calibration object for k=3

KEY INSIGHT: The barrier family should have HARMONIC codegrees across scales — rho_t(T) ~ 1/t — so that fixing one bad witness at scale t only exposes the next at scale t+1. The pair-choice family is too flat (rho_t ~ 2^{-t}). The right adversary needs genuine multiscale structure.

CANDIDATE CONSTRUCTION: Layered ternary-block system with Theta(log w) scales, where each scale contributes one bad witness and conditioning on that witness produces a renormalized copy of the family on remaining scales.

D/(\sigma(\sigma+1)), & 1\le \sigma<L,\\[1mm]

P_{\sigma,b}=P_\sigma\cup\{\ell_{\sigma,b}\},\qquad 1\le b\le m_\sigma.

Each set has size `L+1`.

\[
\sum_{\sigma=j}^{L-1}\frac1{\sigma(\sigma+1)}+\frac1L=\frac1j

gives
\[
\sum_{\sigma=1}^L m_\sigma = D

and, for the prefix witness
\[

\Pr[X_j\subseteq A]=\frac1j.

\[
\rho(X_j)=1/j.

\[
\Pr[X_{j+t}\subseteq A\mid X_j\subseteq A]=\frac{1/(j+t)}{1/j}=\frac{j}{j+t}.
\]
In particular, on dyadic scales `j=2^q`,
\[
\Pr[X_{2^{q+r}}\subseteq A\mid X_{2^q}\subseteq A]=2^{-r}.
\]

So:

- **PROVEN:** this family has the harmonic profile `\rho(X_j)=1/j`.
- **PROVEN:** after linking on `X_j`, the remaining bad-prefix profile is renormalized by `j/(j+t)`.
- **PROVEN:** on dyadic scales, the renormalization is exactly geometric.

This is the first explicit family I know that matches the workshop’s “fix one scale, reveal the next” heuristic.

## 1. A rigorous toy barrier on `Θ(log w)` scales

This realizes exactly the “fix one scale, expose the next” recursion, but without the final regularity property.

### Construction

Fix integers `L >= 1` and `M >= 1`.

For each layer `j = 1,2,\dots,L`, let
\[
U_j := [Mj]\times [j].
\]
For each `b \in [j]`, define the column-pattern
\[
P_{j,b} := [Mj]\times\{b\} \subseteq U_j.
\]
So `|P_{j,b}| = Mj`, and the `j` patterns in layer `j` are pairwise disjoint.

Now define
\[
\mathcal F_{L,M}
:=\Bigl\{\bigcup_{j=1}^L P_{j,b_j} : b_j\in [j]\text{ for all }j\Bigr\}.
\]
Every member has size
\[
W(L,M):=\sum_{j=1}^L Mj = M\frac{L(L+1)}2.
\]
Hence `\mathcal F_{L,M}` is `W(L,M)`-uniform. Its size is
\[
|\mathcal F_{L,M}|=\prod_{j=1}^L j = L!.
\]

If I want exact `w`-uniformity, I just pad every set by the same fixed dummy set `D` of size `w-W(L,M)`. That does not affect any link calculation below.

## 2. Exact self-similarity of links

For each layer `j` and each `b\in[j]`, define the witness
\[
T_{j,b}:=P_{j,b}.
\]

### Lemma 1
For every `j,b`,
\[
\Pr_{A\sim \mathcal F_{L,M}}[T_{j,b}\subseteq A]=\frac1j.
\]

**Proof.**
A set `A` is determined by a choice vector `(b_1,\dots,b_L)` with `b_i\in[i]`. The event `T_{j,b}\subseteq A` is exactly the event `b_j=b`. Since `b_j` is uniform on `[j]`, the probability is `1/j`. ∎

So layer `j` gives a witness of size `Mj` with density exactly `1/j`. This is the discrete-scale analogue of the desired harmonic profile.

### Lemma 2
For every `j,b`, the link
\[
(\mathcal F_{L,M})_{T_{j,b}}
\]
is canonically isomorphic to
\[
\mathcal F_{L-1,M}
\]
after deleting layer `j` and relabeling the remaining layers.

**Proof.**
Conditioning on `T_{j,b}` forces only the `j`th layer choice. All other layer choices remain free and independent, with exactly `i` choices in layer `i`. After removing the forced layer and relabeling indices, one gets the same product construction with one fewer layer. ∎

This is the formal version of `[P4]`: bounded link recursion preserves the obstruction.

---

## 3. No nonempty `C`-spread subfamily exists once one high layer is present

This is stronger than a lower bound for local-link procedures: the obstruction is so rigid that **no nonempty `C`-spread subfamily exists at all**.

Fix a nonempty subfamily `\mathcal G \subseteq \mathcal F_{L,M}`. For each layer `j` and column `b\in[j]`, define
\[
\alpha_{j,b}(\mathcal G)
:= \Pr_{A\sim \mathcal G}[T_{j,b}\subseteq A].
\]
Since every set in `\mathcal G` chooses exactly one column in layer `j`,
\[
\sum_{b=1}^j \alpha_{j,b}(\mathcal G)=1.
\]

### Lemma 3
If `\mathcal G` is `C`-spread, then for every `j,b`,
\[
\alpha_{j,b}(\mathcal G)\le C^{-Mj}.
\]

**Proof.**
`T_{j,b}` has size `Mj`, so `C`-spread means
\[
\Pr[T_{j,b}\subseteq A]\le C^{-|T_{j,b}|}=C^{-Mj}.
\]
That is exactly the claim. ∎

### Theorem 4
If there exists a layer `j` with
\[
j\,C^{-Mj}<1,
\]
then `\mathcal F_{L,M}` has no nonempty `C`-spread subfamily.

**Proof.**
Assume `\mathcal G\subseteq \mathcal F_{L,M}` is nonempty and `C`-spread. Then by Lemma 3,
\[
1=\sum_{b=1}^j \alpha_{j,b}(\mathcal G)\le \sum_{b=1}^j C^{-Mj}=j\,C^{-Mj}<1,
\]
a contradiction. ∎

So for any fixed `C>1`, if `L` is large enough and `M>=1`, then already the top layer violates `C`-spread in every nonempty subfamily.

This is a completely rigorous barrier on `Θ(L)` discrete scales.

## 4. Consequence for local-link regularizers

Take any local-link procedure in your sense. Start from `\mathcal F_{L,M}`.

### Corollary 5
If `L` is large enough that `L C^{-ML}<1`, then any terminating local-link regularization procedure must output the empty family.

**Proof.**
By Theorem 4, no nonempty `C`-spread subfamily of `\mathcal F_{L,M}` exists. Since the procedure terminates only when the current family is `C`-spread, the only possible terminal state is the empty family. ∎

This is stronger than “must lose a lot”: it says the toy barrier is not regularizable at all by any sequence of subfamilies and links, except by deleting everything.

B_{i,j}\qquad (i\in[L],\ j\in[r_i]),
\]
with `|B_{i,j}|=m_i`. Define
\[
H=H(\mathbf r,\mathbf m)
:=\Bigl\{A\subseteq \bigsqcup_{i,j} B_{i,j}:\ |A\cap B_{i,j}|=1\ \forall i,j\Bigr\}.
\]
Every member has size
\[
w=\sum_{i=1}^L r_i,
\]
and
\[
|H|=\prod_{i=1}^L m_i^{r_i}.
\]

This is just a layered product of block-transversal systems, but with the block size depending on the layer. The point is that each layer carries its own codegree scale.

For a witness `T`, write `s_i(T)` for the number of blocks of layer `i` in which `T` chooses exactly one element. If `T` ever contains two elements from the same block, then `T` is incompatible with every `A\in H` and its link is empty. Otherwise:
\[
\Pr_{A\sim H}[T\subseteq A]
=
\prod_{i=1}^L m_i^{-s_i(T)}.
\]
This is immediate from independence across blocks.

So the exact codegree profile is explicit.

Now choose the canonical bad witness at layer `i`:
\[
T_i:=\{x_{i,1},x_{i,2},\dots,x_{i,r_i}\},
\]
where one fixed element `x_{i,j}\in B_{i,j}` is chosen in each layer-`i` block. Then
\[
\Pr[T_i\subseteq A]=m_i^{-r_i}.
\]
Also the link is
\[
H_{T_i}\cong H(r_1,\dots,r_{i-1},0,r_{i+1},\dots,r_L;\ m_1,\dots,m_{i-1},m_{i+1},\dots,m_L),
\]
because conditioning on `T_i` freezes all layer-`i` choices and leaves all other layers untouched. This is the exact self-similarity one wants: linking on a whole offending layer simply deletes that layer and exposes the next one unchanged.

Now I state the corrected discrete-scale barrier.

Let the target spread parameter be a number `\Lambda>0`. Call a family `\Lambda`-spread if
\[
\Pr[T\subseteq A]\le \Lambda^{-|T|}
\qquad\text{for every witness }T.
\]
For the layered family above, `H` is `m_1`-regular in the strong exact sense
\[
\Pr[T\subseteq A]\le m_1^{-|T|},
\]
because each factor is at most `m_1^{-1}`. But if `\Lambda>m_1`, then layer `1` witnesses are too dense; if `\Lambda>m_2`, then after killing layer `1`, layer `2` witnesses are still too dense; and so on.

Here is the precise statement.

**Lemma 1.**  
Let `\Lambda>0`, and let
\[
J(\Lambda):=\{i\in[L]: m_i<\Lambda\}.
\]
Suppose `F\subseteq H(\mathbf r,\mathbf m)` is obtained from `H` by any sequence of operations of the form:
1. replace the current family by a subfamily, or
2. replace the current family by the link on a witness `T`.

If the final family `F` is `\Lambda`-spread and nonempty, then along the sequence one must perform a link operation whose witness intersects every block of each layer `i\in J(\Lambda)`.

**Proof.**  
Fix `i\in J(\Lambda)`. Suppose no link witness ever intersects some block `B_{i,j}`. Then in the final family there is still freedom inside that block, and in fact the canonical witness supported on all still-unfixed blocks of layer `i` has density at least `m_i^{-q}` where `q` is the number of unfixed blocks remaining in that layer. Since `m_i<\Lambda`,
\[
m_i^{-q}>\Lambda^{-q},
\]
so this witness violates `\Lambda`-spread. Therefore every layer with `m_i<\Lambda` must be completely hit by some link witness. ∎

A subfamily operation alone cannot help against this canonical witness, because the statement is only using the existence of unfixed blocks; deleting sets can reduce density, but unless the algorithm spends a factor `m_i` per unfixed block, it cannot beat the exact transversal symmetry. This is quantified next.

**Lemma 2.**  
Let `F\subseteq H(\mathbf r,\mathbf m)`. Fix a layer `i`. If no link operation has fixed any element in any block of layer `i`, and `F` is `\Lambda`-spread for some `\Lambda>m_i`, then
\[
|F|\le \Bigl(\frac{m_i}{\Lambda}\Bigr)^{r_i}|H|.
\]

**Proof.**  
For each `A\in F`, let `A_i:=A\cap (\bigsqcup_{j=1}^{r_i} B_{i,j})`. There are exactly `m_i^{r_i}` possible layer-`i` patterns. Averaging gives a pattern `u` with
\[
|\{A\in F:A_i=u\}|\ge |F|/m_i^{r_i}.
\]
Let `T_u` be the corresponding canonical witness of size `r_i`. Then
\[
\Pr_{A\sim F}[T_u\subseteq A]\ge \frac{1}{m_i^{r_i}}.
\]
If `F` is `\Lambda`-spread, this probability is at most `\Lambda^{-r_i}`. Hence
\[
\frac{|F|}{|H|}\le \Bigl(\frac{m_i}{\Lambda}\Bigr)^{r_i}.
\]
∎

This is the key quantitative barrier: to make layer `i` harmless without linking it away, one must lose a factor at least `(m_i/\Lambda)^{r_i}`.

Now choose concrete parameters. Let
\[
L:=\lfloor \tfrac12\log_2 w\rfloor,\qquad
m_i:=2^i,\qquad
r_i:=\left\lfloor \frac{w}{L}\right\rfloor
\]
for all `i`, adjusting the last layer by at most `L` so that `\sum_i r_i=w`. Then:
\[
|H|=\prod_{i=1}^L 2^{i r_i}
=2^{(\sum_i i r_i)}
= \exp\!\bigl(\Theta(wL)\bigr)
= \exp\!\bigl(\Theta(w\log w)\bigr).
\]
Also `H` is `2`-regular, hence certainly `\kappa`-regular for every `\kappa\ge2`; in particular it fits the “low regularity” side of the obstruction, though not the original impossible `(1)+(2)` combination.

Now set the target
\[
\Lambda:=2^{L+1}\asymp w.
\]
Then every layer is below target: `m_i<\Lambda` for all `i`. By Lemma 1, every local-link regularizer must eliminate every layer. By Lemma 2, if it tries to avoid linking layer `i`, it must lose a factor at least
\[
\Bigl(\frac{2^i}{2^{L+1}}\Bigr)^{r_i}=2^{-(L+1-i)r_i}.
\]
Summing across layers gives total loss at least
\[
2^{-\sum_{i=1}^L (L+1-i)r_i}
=
\exp\!\bigl(-\Theta(wL)\bigr)
=
\exp\!\bigl(-\Theta(w\log w)\bigr).
\]
Equivalently, any successful procedure must discard all but an `\exp(-\Theta(w\log w))` fraction of the family, unless it performs the full sequence of links deleting all `L` layers one by one. Either way the size loss is `\exp(\Theta(w\log w))`.

So this gives a rigorous `Theta(log w)`-scale barrier, but for the repaired target “raise spread from `Theta(1)`/`Theta(log w)` to `\Lambda\asymp w`” rather than the impossible fixed-constant target.

There is also a harmonic variant. If instead one takes
\[
m_i=i+1,
\]
then
\[
\Pr[T_i\subseteq A]=(i+1)^{-r_i},
\]
so the effective inverse-density scale is linear in `i`, which is the exact discrete analogue of the desired harmonic profile. The same proofs go through with `\Lambda=L+2`, and now
\[
|H|=\prod_{i=1}^L (i+1)^{r_i}
=\exp\!\Bigl(\frac{w}{L}\sum_{i=1}^L \log(i+1)\Bigr)
=\exp\!\bigl(\Theta(w\log L)\bigr).
\]
For `L=\Theta(\log w)`, this is
\[
\exp\!\bigl(\Theta(w\log\log w)\bigr).
\]
Thus the harmonic choice gives exactly the weaker lower bound one would expect from “one payment per logarithmic scale”.

That is the clean split:

1. Exponential scales `m_i=2^i` give `\exp(\Theta(w\log w))` loss.
2. Harmonic scales `m_i=i+1` give `\exp(\Theta(w\log\log w))` loss and better match the heuristic `\rho_t\sim 1/t`.

## Theorem 1: Exact performance of the canonical scalar-target regularizer

**Claim.**
For every `\Lambda>=2`, the algorithm `\mathcal A_\Lambda` outputs a nonempty `\Lambda`-spread family `F^*`, and
\[
\frac{|F^*|}{|H(\mathbf r,\mathbf m)|}
=
\prod_{i\in S(\Lambda)} m_i^{-r_i}.
\]

### Proof

By `[P2]`, linking on `T_i` deletes layer `i` and leaves an isomorphic copy of the product family on the remaining layers. Therefore after linking every layer in `S(\Lambda)`, the final family is exactly
\[
F^*\cong H\bigl((r_i)_{i\notin S(\Lambda)},(m_i)_{i\notin S(\Lambda)}\bigr),
\]
hence
\[
|F^*|=\prod_{i\notin S(\Lambda)} m_i^{r_i},
\]
so
\[
\frac{|F^*|}{|H|}=\prod_{i\in S(\Lambda)} m_i^{-r_i}.
\]

It remains to prove `\Lambda`-spread.

Take any witness `T`. If `T` contains two points from one remaining block, then `\Pr[T\subseteq A]=0`. Otherwise, by `[P1]`,
\[
\Pr_{A\sim F^*}[T\subseteq A]
=
\prod_{i\notin S(\Lambda)} m_i^{-s_i(T)},
\]
where `s_i(T)` counts the number of remaining layer-`i` blocks hit once by `T`.

Since `i\notin S(\Lambda)` implies `m_i\ge \Lambda`, we have
\[
m_i^{-s_i(T)}\le \Lambda^{-s_i(T)}.
\]
Multiplying over `i\notin S(\Lambda)` gives
\[
\Pr[T\subseteq A]\le \Lambda^{-\sum_{i\notin S(\Lambda)} s_i(T)}=\Lambda^{-|T|}.
\]
So `F^*` is `\Lambda`-spread. ∎

The constraint matrix `M(a,b,c)` has rank `2` as soon as at least two of the three column types are present, because any two of
\[
e_1,\ e_2,\ e_1+e_2
\]
are linearly independent over `F_q`. Hence the solution space of (1) has dimension `a+b+c-2`. Each solution gives one member of the family. ∎

**Lemma 2.** Let `T` be any feasible witness. Then the link `(H_{a,b,c}(q))_T` is isomorphic, by independent label-shifts on the remaining blocks, to `H_{a',b',c'}(q)`.

**Proof.**
Fix one completion `x^0` of `T`. Any other completion has the form `x^0+y`, where `y` vanishes on the fixed coordinates and satisfies the homogeneous residual system
\[
\sum_{u\in U'} y_u+\sum_{w\in W'} y_w=0,
\qquad
\sum_{v\in V'} y_v+\sum_{w\in W'} y_w=0,
\]
with `U',V',W'` the unfixed coordinates. Thus the completion set is an affine translate of the kernel defining `H_{a',b',c'}(q)`. Translating labels in each remaining block by `-x^0_z` gives an isomorphism of set families. ∎

This is the exact arbitrary-witness renormalization that the layered product obstruction did **not** have: every feasible link stays inside the same 3-parameter class.

## 2. Exact codegree formula

Let
\[
r(a',b',c') :=
\begin{cases}
2,& \text{if at least two of }a',b',c'\text{ are positive},\\
1,& \text{if exactly one of }a',b',c'\text{ is positive},\\
0,& \text{if }a'=b'=c'=0.
\end{cases}
\]

**Lemma 3.** For every feasible witness `T`,
\[
\Pr_{A\in H_{a,b,c}(q)}[T\subseteq A]
=
q^{-\,(\alpha+\beta+\gamma-2+r(a',b',c'))}.
\tag{2}
\]

Equivalently, with `s:=|T|=\alpha+\beta+\gamma`,
\[
\lambda(T)=
\begin{cases}
q^{-s},& \text{if at least two direction classes survive after fixing }T,\\
q^{-(s-1)},& \text{if exactly one direction class survives},\\
q^{-(s-2)},& \text{if }T\text{ fixes all coordinates}.
\end{cases}
\tag{3}
\]

**Proof.**
By Lemma 2, the number of completions of `T` equals the size of `H_{a',b',c'}(q)`, namely `q^{a'+b'+c'-r(a',b',c')}`. Dividing by
\[
|H_{a,b,c}(q)|=q^{a+b+c-2}
\]
gives
\[
\lambda(T)
= q^{a'+b'+c'-r(a',b',c')-(a+b+c-2)}
= q^{-[(a-a')+(b-b')+(c-c')-2+r(a',b',c')]}
\]
which is exactly (2). ∎

So the witness probability depends not only on `|T|`, but on the **residual type** `(a',b',c')`.

## 4. Finite-state renormalization graph

The codegree exponent of that move is determined by the rank of the *new* state through (2).

#### The Formalizer (codex/gpt-5.4)

I am formalizing the linear-family candidate from `[P22]`–`[P24]` into an exact state machine. This does not solve `O13/O14`, but it turns them into a precise representable-matroid problem and gives the monotonicity/shift identities in a form we can actually use.

Fix a full-row-rank matrix
\[
G\in \mathbb F_q^{r\times E},\qquad |E|=w,
\]
and let `H(G,q)` be the associated `w`-uniform transversal family:
\[
H(G,q):=\{A_x=\{(e,x_e):e\in E\}: x\in \ker G\}.
\]
Let `M=M(G)` be the column matroid of `G`, and let `N:=M^*` be its dual.

For a feasible witness `T`, write `\supp(T)\subseteq E` for the set of coordinates fixed by `T`. Define
\[
\phi_G(S):=-\log_q \Pr_{A\in H(G,q)}[S\subseteq A].
\]

I will prove that `\phi_G` is exactly the dual rank function.

## Theorem 1. Exact dual-rank formula

For every feasible witness support `S\subseteq E`,
\[
\phi_G(S)=r_N(S).
\tag{1}
\]

### Proof
From `[The Builder, pass 3]`, for every feasible witness support `S`,
\[
\Pr[S\subseteq A]
=
q^{-(|S|-r_M(E)+r_M(E\setminus S))}.
\]
Hence
\[
\phi_G(S)=|S|-r_M(E)+r_M(E\setminus S).
\]
But the dual-rank formula for a matroid says
\[
r_{M^*}(S)=|S|-r_M(E)+r_M(E\setminus S).
\]
Therefore
\[
\phi_G(S)=r_{M^*}(S)=r_N(S).
\]
This proves (1). ∎

So the witness exponent is not merely “rank-drop-like”; it is literally a matroid rank function.

## Corollary 2. Monotonicity and submodularity

For all `A,B\subseteq E`,
\[
\phi_G(A)\le \phi_G(B)\qquad\text{whenever }A\subseteq B,
\tag{2}
\]
and
\[
\phi_G(A)+\phi_G(B)\ge \phi_G(A\cup B)+\phi_G(A\cap B).
\tag{3}
\]

### Proof
By Theorem 1, `\phi_G=r_N`, and matroid rank functions are monotone and submodular. ∎

This is the clean formal version of the monotonicity property we wanted.

## Theorem 3. Exact shift under links

Let `X\subseteq E` be a feasible witness support, and link on `X`. The resulting family is isomorphic to `H(G|_{E\setminus X},q)` up to label shifts, hence its new exponent function is
\[
\phi_{G/X}(Y)=\phi_G(X\cup Y)-\phi_G(X)
\qquad (Y\subseteq E\setminus X).
\tag{4}
\]

Equivalently,
\[
\phi_{G/X}(Y)=r_{N/X}(Y).
\tag{5}
\]

### Proof
The affine-renormalization argument from `[P22]` gives that after fixing `X`, the residual family is the kernel family of the deleted column configuration `G|_{E\setminus X}`.

Applying Theorem 1 to the residual family,
\[
\phi_{G/X}(Y)=r_{(M|_{E\setminus X})^*}(Y).
\]
But duality gives
\[
(M|_{E\setminus X})^*=M^*/X=N/X,
\]
so
\[
\phi_{G/X}(Y)=r_{N/X}(Y).
\]
Now use the contraction rank formula:
\[
r_{N/X}(Y)=r_N(X\cup Y)-r_N(X).
\]
By Theorem 1 again,
\[
r_N(X\cup Y)-r_N(X)=\phi_G(X\cup Y)-\phi_G(X).
\]
This proves (4) and (5). ∎

This is the exact shift identity. It is stronger than the informal “bad witness at one scale exposes the next”: in the linear model, link dynamics is literally matroid contraction.

## Corollary 4. Exact deficit shift for `C`-spread

Fix `C>1` and set
\[
\alpha:=\log_q C.
\]
Define the `C`-spread deficit
\[
\Delta_{N,\alpha}(S):=\alpha |S|-r_N(S).
\tag{6}
\]
Then `H(G,q)` is `C`-spread iff
\[
\Delta_{N,\alpha}(S)\le 0\qquad\text{for all }S\subseteq E.
\tag{7}
\]

After linking on `X`,
\[
\Delta_{N/X,\alpha}(Y)=\Delta_{N,\alpha}(X\cup Y)-\Delta_{N,\alpha}(X)
\qquad (Y\subseteq E\setminus X).
\tag{8}
\]

### Proof
By Theorem 1,
\[
\Pr[S\subseteq A]=q^{-r_N(S)}.
\]
The condition `\Pr[S\subseteq A]\le C^{-|S|}` is therefore
\[
q^{-r_N(S)}\le q^{-\alpha |S|},
\]
equivalently
\[
r_N(S)\ge \alpha |S|,
\]
which is exactly (7).

For the shift,
\[
\Delta_{N/X,\alpha}(Y)
=
\alpha|Y|-r_{N/X}(Y)
=
\alpha|Y|-\bigl(r_N(X\cup Y)-r_N(X)\bigr)
\]
by Theorem 3. Rearranging,
\[
\Delta_{N/X,\alpha}(Y)
=
\bigl(\alpha|X\cup Y|-r_N(X\cup Y)\bigr)-\bigl(\alpha|X|-r_N(X)\bigr),
\]
which is (8). ∎

So in the linear model, the right state variable is not the raw probability profile but the dual-rank deficit function, and link steps act by exact subtraction.

## Corollary 5. Exact cumulative loss for link-only algorithms

Suppose a local-link procedure uses only link steps, on witness supports
\[
X_1,\dots,X_t
\]
chosen sequentially in the residual instances. Let
\[
U:=X_1\sqcup X_2\sqcup\cdots\sqcup X_t
\]
viewed back in the original ground set. Then the final family size satisfies
\[
\frac{|F_t|}{|F_0|}=q^{-r_N(U)}.
\tag{9}
\]

In particular, the total multiplicative loss depends only on the final contracted set `U`, not on the order of link steps.

### Proof
A single link on `X` multiplies size by
\[
q^{-\phi_G(X)}=q^{-r_N(X)}.
\]
At step `j`, in the residual matroid `N/(X_1\cup\cdots\cup X_{j-1})`, the loss factor is
\[
q^{-r_{N/(X_1\cup\cdots\cup X_{j-1})}(X_j)}.
\]
By contraction rank,
\[
r_{N/(X_1\cup\cdots\cup X_{j-1})}(X_j)
=
r_N(X_1\cup\cdots\cup X_j)-r_N(X_1\cup\cdots\cup X_{j-1}).
\]
Multiplying the losses telescopes:
\[
\prod_{j=1}^t
q^{-\bigl(r_N(X_1\cup\cdots\cup X_j)-r_N(X_1\cup\cdots\cup X_{j-1})\bigr)}
=
q^{-r_N(U)}.
\]
This is (9). ∎

This gives an exact optimization problem for the linear model:

> Find the minimum dual rank of a set `U` such that the contraction `N/U` satisfies  
> `r_{N/U}(S)\ge \alpha |S|` for all `S`.

That is the precise link-only regularization problem.

## Proposition 7. Any fully size-homogeneous linear obstruction collapses to a uniform matroid

Assume for a `q`-representable matroid `N` that
\[
r_N(S)=f(|S|)\qquad\text{for all }S\subseteq E
\]
for some function `f:\{0,\dots,w\}\to \mathbb Z_{\ge 0}`. Then
\[
f(t)=\min(t,d)
\]
for some `d`, i.e. `N` is a uniform matroid.

### Proof
Since `r_N` is a matroid rank function, every marginal
\[
f(t+1)-f(t)
\]
must lie in `\{0,1\}`: adding one element increases rank by at most `1`, and by symmetry the increment depends only on `t`.

Submodularity implies diminishing returns. For any `A\subset B` and `e\notin B`,
\[
r_N(A\cup\{e\})-r_N(A)\ge r_N(B\cup\{e\})-r_N(B).
\]
Since the marginal depends only on set size, this says
\[
f(t+1)-f(t)\ge f(t+2)-f(t+1)
\]
for every `t`. Thus the `0/1` increment sequence is nonincreasing, so it has the form
\[
1,1,\dots,1,0,0,\dots,0.
\]
Hence for some `d`,
\[
f(t)=\min(t,d).
\]
That is exactly the rank function of the uniform matroid `U_{d,w}`. ∎

So within the linear model, “hide the geometry completely so witnesses of the same size look identical” forces the trivial uniform case. The only way to get multiscale behavior is through genuine same-size rank splitting, i.e. typed witness geometry.

#### The Adversary (codex/gpt-5.4)

I am attacking `O15/O17` from the opposite direction: instead of trying to build a `q`-representable obstruction, I will show that the entire `q`-linear route is quantitatively capped at `exp(O(w log log w))` once `\kappa=\Theta(\log w)` is imposed. Let `G\in \mathbb F_q^{r\times E}` with `|E|=w`, and let
\[
H(G,q):=\{A_x=\{(e,x_e):e\in E\}:x\in\ker G\}.
\]
Let `M=M(G)` and `N=M^*`. I use `[P25]` and `[P28]` without re-deriving them.

**Theorem 1 (field-size ceiling for linear obstructions).**
Assume `H(G,q)` is `\kappa`-regular with `\kappa>1`, and
\[
|H(G,q)|\ge (c\log w)^w
\]
for some absolute `c>0`. Then:

1. `q\le \kappa`.
2. `q\ge c\log w`.
3. Hence if `\kappa=\Theta(\log w)`, then necessarily `q=\Theta(\log w)`.
4. Every link-only local regularization path on `H(G,q)` has multiplicative size loss at most
\[
q^w=\exp(O(w\log\log w)).
\]

So no `q`-representable family with `\kappa=\Theta(\log w)` can force `\exp(\Omega(w\log w))` loss under link-only local regularization.

**Proof.**

Step 1: prove `q\le \kappa`.

Fix `e\in E`. Since `\kappa>1`, no singleton witness can occur with probability `1`, so `r_N(\{e\})\neq 0`. By `[P25]`,
\[
\Pr[(e,t)\subseteq A]=q^{-r_N(\{e\})}
\]
for every feasible singleton value `t`. Because matroid rank on a singleton is either `0` or `1`, we must have
\[
r_N(\{e\})=1.
\]
Hence every feasible singleton witness has probability exactly `q^{-1}`. Since the family is `\kappa`-regular,
\[
q^{-1}\le \kappa^{-1},
\]
so `q\le \kappa`.

Step 2: prove `q\ge c\log w`.

We have
\[
|H(G,q)|=|\ker G|=q^{w-r_M(E)}=q^{r_N(E)}.
\]
Since `r_N(E)\le w`,
\[
|H(G,q)|\le q^w.
\]
Combining with the size lower bound,
\[
(c\log w)^w\le |H(G,q)|\le q^w,
\]
hence
\[
q\ge c\log w.
\]

Step 3: combine.

If `\kappa=\Theta(\log w)`, then Step 1 gives `q=O(\log w)` and Step 2 gives `q=\Omega(\log w)`, so
\[
q=\Theta(\log w).
\]

Step 4: bound every link-only loss.

By `[P28]`, for any link-only procedure whose total contracted support is `U\subseteq E`,
\[
\frac{|F_0|}{|F_t|}=q^{r_N(U)}.
\]
Since `r_N(U)\le |U|\le w`,
\[
\frac{|F_0|}{|F_t|}\le q^w.
\]
Using Step 3,
\[
q^w=(\Theta(\log w))^w=\exp(O(w\log\log w)).
\]
This proves the claim. ∎

This has an immediate consequence for the current open linear-matroid program.

**Corollary 2.**
Any attempt to solve `O15` or `O17` by a `q`-representable family with `\kappa=\Theta(\log w)` cannot produce an `\exp(\Omega(w\log w))` lower bound against link-only local regularizers.

**Proof.**
Theorem 1 shows that along every link-only path the total possible multiplicative loss is at most `\exp(O(w\log\log w))`. Therefore no lower bound of order `\exp(\Omega(w\log w))` can hold in that model. ∎

This is not just a weakness of the rank-2 example. It kills the whole representable-matroid route at the scale we want.

There is also a sharper interpretation in the dual-matroid language. Since
\[
|H(G,q)|=q^{r_N(E)},
\]
the total “loss budget” available to any link-only algorithm is at most the entire kernel dimension times `\log q`. Under `\kappa=\Theta(\log w)`, that budget is only `\Theta(w\log\log w)`. So even a perfect multiscale contraction state machine cannot pay more than `w\log\log w` in the exponent.

That means the only remaining hope for an `\exp(\Omega(w\log w))` barrier is to leave the representable-kernel framework, or to use subfamily restrictions in an essentially nonlinear way that is not reducible to contraction rank.

\]

## 1. THEOREM/RESULT STATEMENT

### Theorem A (the original theorem is false as stated)
Let `C>1` be any fixed constant. Suppose a `w`-uniform family `H_w` satisfies

\[
\Pr_{A\sim H_w}[T\subseteq A]\le \kappa(w)^{-|T|}
\qquad\text{for every witness }T,
\]

with `\kappa(w)=\Theta(\log w)`. Then for all sufficiently large `w`, the family `H_w` is already `C`-spread:

\[
\Pr_{A\sim H_w}[T\subseteq A]\le C^{-|T|}
\qquad\text{for every }T.
\]

Consequently, the “local-link regularization” procedure may terminate immediately at time `0` with no loss. Therefore the workshop’s target theorem,

> there exist `w`-uniform families `H_w` with `|H_w|\ge (c\log w)^w`, satisfying `\kappa=\Theta(\log w)` regularity, but such that **every** local-link regularization to fixed-constant `C`-spread loses `\exp(\Omega(w\log w))`,

is false.

---

### Theorem B (ceiling for the linear-kernel / representable-matroid route)
Let `G\in\mathbb F_q^{r\times E}` with `|E|=w`, and define the `w`-uniform transversal kernel family

\[
H(G,q):=\Bigl\{A_x=\{(e,x_e):e\in E\}:x\in\ker G\Bigr\}.
\]

Assume:

1. `H(G,q)` is `\kappa`-regular with `\kappa>1`,
2. `|H(G,q)|\ge (c\log w)^w` for some absolute `c>0`.

Then:

1. `q\le \kappa`,
2. `q\ge c\log w`,
3. hence if `\kappa=\Theta(\log w)`, then necessarily `q=\Theta(\log w)`.

Moreover, for any local-link procedure using only link steps, the total multiplicative size loss is at most

\[
q^w=\exp(O(w\log\log w)).
\]

Therefore no `q`-representable / link-only construction can prove an `\exp(\Omega(w\log w))` lower bound at the size scale `|H_w|\ge (c\log w)^w` and regularity scale `\kappa=\Theta(\log w)`.

---

### Theorem C (exact structure of link dynamics in linear kernel families)
With `H(G,q)` as above, let `M=M(G)` be the column matroid of `G`, and `N=M^*` its dual. For a witness support `S\subseteq E`,

\[
\Pr_{A\sim H(G,q)}[S\subseteq A]=q^{-r_N(S)}.
\]

If one links on a feasible witness support `X`, then the new exponent function is exactly dual-matroid contraction:

\[
-\log_q \Pr_{\text{new}}[Y\subseteq A]
=
r_{N/X}(Y)
=
r_N(X\cup Y)-r_N(X).
\]

Thus, in the linear model, local-link dynamics is exactly contraction in a dual matroid.

\qquad (w\ge w_0).
\]

Fix `C>1`. For all sufficiently large `w`, we have `a\log w\ge C`, hence `\kappa(w)\ge C`. Therefore for every witness `T`,

\[
\Pr[T\subseteq A]\le \kappa(w)^{-|T|}\le C^{-|T|}.
\]

So `H_w` is already `C`-spread. A valid local-link regularization is: start with `F_0=H_w`, observe no offending witness exists, terminate. Loss factor: `1`.

This contradicts any claim that every such procedure must lose `\exp(\Omega(w\log w))`. ∎

### Proof of Theorem C
Fix `S\subseteq E`. To require `S\subseteq A_x` is to fix the coordinates `x_e` on `S` to prescribed values. The number of completions equals the number of solutions to the residual linear system on `E\setminus S`, namely

\[
q^{|E\setminus S|-r_M(E\setminus S)}.
\]

Since

\[
|H(G,q)|=|\ker G|=q^{|E|-r_M(E)},
\]

we get

\[
\Pr[S\subseteq A]
=
q^{-(|S|-r_M(E)+r_M(E\setminus S))}.
\]

Using the dual-rank identity,

\[
r_{M^*}(S)=|S|-r_M(E)+r_M(E\setminus S),
\]

this becomes

\[
\Pr[S\subseteq A]=q^{-r_N(S)}.
\]

Now link on `X`. The residual family is an affine translate of the kernel family on the remaining columns, so its dual-rank function is that of `N/X`. Therefore for `Y\subseteq E\setminus X`,

\[
-\log_q \Pr_{\text{new}}[Y\subseteq A]=r_{N/X}(Y).
\]

Finally, contraction rank satisfies

\[
r_{N/X}(Y)=r_N(X\cup Y)-r_N(X).
\]

This proves the exact shift formula. ∎

### Proof of Theorem B
We use Theorem C.

#### Step 1: `q\le \kappa`
Take any element `e\in E`. Since `\kappa>1`, no feasible singleton witness can have probability `1`, so `r_N(\{e\})\neq 0`. Since singleton matroid rank is either `0` or `1`, we must have `r_N(\{e\})=1`. Hence every feasible singleton has probability exactly

\[
q^{-1}.
\]

By `\kappa`-regularity,

\[
q^{-1}\le \kappa^{-1},
\]

so `q\le \kappa`.

#### Step 2: `q\ge c\log w`
We have

\[
|H(G,q)|=q^{r_N(E)}\le q^w.
\]

If `|H(G,q)|\ge (c\log w)^w`, then

\[
q^w\ge (c\log w)^w,
\]

hence `q\ge c\log w`.

Combining Steps 1 and 2, if `\kappa=\Theta(\log w)`, then necessarily `q=\Theta(\log w)`.

#### Step 3: link-only loss ceiling
Suppose a link-only algorithm contracts total support `U`. By the exact contraction formula, each link step contributes a factor that telescopes, giving final loss exactly

\[
\frac{|F_0|}{|F_t|}=q^{r_N(U)}\le q^{|U|}\le q^w.
\]

Since `q=\Theta(\log w)`, this is at most

\[
(\Theta(\log w))^w=\exp(O(w\log\log w)).
\]

So the linear-kernel / link-only route cannot force `\exp(\Omega(w\log w))`. ∎

## 3. WHAT WAS ACHIEVED

This workshop did **not** prove the original theorem. Instead, it achieved three rigorous outcomes.

1. It disproved the theorem as stated.
   The obstruction is immediate: `\kappa=\Theta(\log w)` regularity is already stronger than fixed-constant `C`-spread for all large `w`.

2. It completely analyzed the linear-kernel route.

At the target size and regularity scale this becomes only `\exp(O(w\log\log w))`, so that route cannot yield the desired `\exp(\Omega(w\log w))`.

3. It identified a genuine finite-state phenomenon beyond size-homogeneity.
   The explicit non-product family `H_{a,b,c}(q)` shows that two witnesses of the same size can have different probabilities and renormalize to different successor states. So any viable barrier must preserve witness geometry; it cannot depend only on witness size.

So the strongest honest conclusion is:

- **Complete disproof** of the original statement.
- **Complete impossibility result** for one major repair route (representable linear kernel families with link-only dynamics).
- **Partial structural progress** toward a corrected problem: the next candidate must be genuinely nonlinear and geometry-sensitive.

## 5. KEY INSIGHT

The single most important insight is:

> **Any successful barrier must encode obstruction in witness geometry, not just witness size, and it must do so in a genuinely nonlinear way.**

Size-only regularity is too strong and kills the original theorem. Product constructions can be regularized by coordinatewise shrinking. Linear kernel constructions reduce exactly to dual matroid contraction and have only `\exp(O(w\log\log w))` total loss budget at `\kappa=\Theta(\log w)`.

So the only plausible remaining path is a hidden multiscale system where:
- witnesses of the same size come in different geometric types,
- local links act by state transitions rather than simple contraction,
- and no cheap global restriction collapses all bad scales at once.
