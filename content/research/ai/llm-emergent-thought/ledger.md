## PROVEN RESULTS (use these, do not re-derive)
- [P1] E_rep strictly refines the earlier emergence score by adding a frame-distance term that separates same-frame surprise from representational novelty. (by The Cognitive Scientist, iter 1; merges [P1], [P5]) (by compressed, iter 0)
- [P2] In a greedy frame-selection model, a hard constraint can create a phase transition from an overlearned frame \(R_0\) to a more fertile frame \(R_1\) whenever the ban lowers effective salience below the alternative frame’s salience. (by The Cognitive Scientist, iter 1; merges [P2], [P6]) (by compressed, iter 0)
- [P3] If immature useful ideas satisfy \(p_t > q_t\) at each stage, then incubation-before-verification preserves more long-maturation ideas than verify-first. (by The Cognitive Scientist, iter 1; merges [P3], [P7]) (by compressed, iter 0)
- [P4] If \(h\) is unreachable in the closure \(\mathrm{Reach}(R_0)\) of low-cost moves in the default frame but reachable in a transformed frame \(R_1\), then generating \(h\) requires a frame change; it is not merely same-frame recombination. (by The Cognitive Scientist, iter 1; merges [P4], [P8]) (by compressed, iter 0)
- [P5] If frames induce separated distributions over a typed action trace and the final hypothesis is more salient in its true frame, then the terminal frame and hence \(C(R \to h)\) are automatically estimable from trace data with exponentially decaying error. (by The ML Theorist, iter 1) (by compressed, iter 0)
- [P6] In a one-switch model starting from the default frame, the switch time and switched-to frame are jointly recoverable from traces with error exponentially small in the post-switch segment length. (by The ML Theorist, iter 1) (by compressed, iter 0)
- [P7] If the local incubation gain \(R_t = p_t q_{t+1}/q_t\) is strictly decreasing in \(t\), then the optimal switch from object-grade incubation to theorem-grade verification is a sharp threshold, because the expected yield \(Y(u)=\left(\prod_{s<u}p_s\right)q_u\) is unimodal. (by The Harness Engineer, iter 1) (by compressed, iter 0)
- [P8] If two idea families satisfy \(R_t^A<1<R_t^B\) at the same stage, then no single global verifier schedule can be optimal for both; a routed two-lane policy strictly dominates any global stage choice on every positive mixture of the two families. (by The Harness Engineer, iter 1) (by compressed, iter 0)
- [P9] If a learned typed action alphabet has cells on which pairwise frame log-likelihood ratios vary by at most \(\varepsilon\), then its coarse-grained frame KL loses at most \(\varepsilon\); therefore automatic action learning can preserve novelty measurability. (by The Cognitive Scientist, iter 1) (by compressed, iter 0)
- [P10] If the learned abstraction \(\phi\) retains too little mutual information with the underlying frame, then no decoder can reliably recover frame identity or frame change from the abstracted trace; this is an information-theoretic impossibility bound via Fano. (by The Cognitive Scientist, iter 1) (by compressed, iter 0)
- [P11] A positive-cost frame switch is worthwhile exactly when it raises accessible target-family mass enough that \(1-(1-\mu_1)^{B-c}\) exceeds the no-switch ceiling \(1-(1-\mu_0)^B\); for small masses this is \((B-c)\mu_1 \gg B\mu_0\). (by The ML Theorist, iter 1; summarizes [P15], [P16], [P17], [P30]) (by compressed, iter 0)
- [P12] Multi-model diversity gives no qualitative novelty gain when models’ target-family mass vectors are approximately collinear; complementarity matters only through non-collinear mass on different target subfamilies. (by The ML Theorist, iter 1) (by compressed, iter 0)
- [P13] Logging object hits and mature descendants is sufficient to identify empirical proposal mass and maturation success, with exact decomposition \(\widehat Y=\widehat\mu\,\widehat s\). (by The Harness Engineer, iter 1; merges [P19], [P20], [P21], [P23]) (by compressed, iter 0)
- [P14] For fixed intervention lane, proposal mass \(\mu_I\) does not affect the optimal within-lane incubation depth; that depth is governed by the maturation profile \(R_{I,t}\). (by The Harness Engineer, iter 1) (by compressed, iter 0)
- [P15] For a typed action trace, the best two-frame-vs-one-frame code-length gain at split \(\tau\) is exactly \(T\,\mathrm{JSD}_\alpha(\hat p_{1:\tau},\hat p_{\tau+1:T})\). (by The Cognitive Scientist, iter 1) (by compressed, iter 0)
- [P16] In the one-switch multinomial trace model, a penalized MDL detector with \(\lambda_T\to\infty\) and \(\lambda_T/T\to 0\) consistently rejects false switches and detects true frame changes, with consistent localization of the switch time. (by The Cognitive Scientist, iter 1) (by compressed, iter 0)
- [P17] Shared cross-pollination contamination \(P_\gamma=(1-\gamma)P+\gamma W,\ Q_\gamma=(1-\gamma)Q+\gamma W\) contracts frame-identifiability. (by The Cognitive Scientist, iter 1) (by compressed, iter 0)
- [P18] A hard constraint improves target-family proposal mass iff the banned set is less target-enriched than the ambient distribution. (by The ML Theorist, iter 1; pairs with [P27]) (by compressed, iter 0)
- [P19] If one model componentwise dominates another on per-family theorem mass \(\nu_{k,j}\), then allocating compute to the dominated model never improves Bayes-averaged novelty success. (by The ML Theorist, iter 1) (by compressed, iter 0)
- [P20] Under randomized lane assignment, object-hit rate \(\mu_\ell\), maturation success \(s_\ell\), and false-claim leakage \(f_\ell\) are separately causally identifiable from logged attempt data. (by The Harness Engineer, iter 1) (by compressed, iter 0)
- [P21] If the marginal yield-to-leakage ratio \(\Lambda(u)=\Delta A(u)/\Delta B(u)\) decreases with incubation depth, then the optimal verification handoff is a sharp threshold. (by The Harness Engineer, iter 1; merges [P33], [P34]) (by compressed, iter 0)

## MECHANISMS (conserved quantities, forcing principles that survived stress testing)

- [M2] Productive Diversity = Frame Novelty × Translation Fidelity (survived: The Cognitive Scientist, iter 0)
- [M3] Attractor Suppression — creative constraints work when they demote an overlearned representation below a weaker but more fertile one. (survived: The Cognitive Scientist, iter 0)
- [M4] Productive Diversity = Frame Novelty × Translation Fidelity. (survived: The Cognitive Scientist, iter 0)
- [M5] Frame Identifiability Gap — novelty is measurable only when different representations induce statistically separable distributions over reasoning moves and object salience. (survived: The ML Theorist, iter 0)
- [M6] Maturation Dividend — creative interventions help exactly when they raise theoremizability faster than they destroy line survival, i.e. when they increase p_t q_{t+1}/q_t. (survived: The Harness Engineer, iter 0)
- [M7] Diagnostic Compression — a trace abstraction is creativity-relevant iff it compresses within-frame variation while preserving between-frame likelihood ratios. (survived: The Cognitive Scientist, iter 0)
- [M8] Proposal-Mass Barrier — inference-time compute helps novelty only in proportion to the proposal mass that the current representational regime assigns to the target object family. (survived: The ML Theorist, iter 0)
- [M9] Yield Factorization Law. (survived: The Harness Engineer, iter 0)
- [M10] Yield Factorization Law — novelty yield is proposal-mass on the right object family times maturation survival of that family. (survived: The Harness Engineer, iter 0)
- [M11] Common-Frame Contamination — auto-injected shared framing can increase local fluency while linearly shrinking the measurable distance between representations, thereby masking genuine frame change. (survived: The Cognitive Scientist, iter 0)
- [M12] Attractor Reweighting. (survived: The ML Theorist, iter 0)
- [M13] Attractor Reweighting — creative interventions help by deleting or diluting high-mass non-target basins and reallocating mass toward low-salience target families. (survived: The ML Theorist, iter 0)
- [M14] Proto-Idea Option Value — incubation is valuable exactly while one more protected step buys more true-yield than false-leakage cost. The verification trap is a marginal-ratio problem, not a binary “strict vs permissive” choice. (survived: The Harness Engineer, iter 0)
