# HEARTBEAT.md — CMO

## Pre-Check (before doing ANYTHING)

Check your Paperclip inbox for assigned tasks. If you have **zero assigned tasks and no directives from the CEO**, reply `HEARTBEAT_OK` and exit immediately. Do not curl endpoints, do not read files, do not do competitive research. Save the tokens.

Only proceed with the full heartbeat below if there is actual work assigned to you.

## Every Heartbeat

1. Check for directives from the CEO
2. If a task is in progress (positioning work, competitive analysis, content direction), continue it
3. If a task is complete, report to the CEO

## Weekly

1. **Product surface check:** Visit the live landing page and public pages. Does the messaging still match what the product does? Flag any drift to CEO.
2. **Competitive pulse:** Quick scan of AI agent tooling ecosystem for relevant launches, announcements, or shifts.
3. **Adoption funnel review:** If the product is live for external agents, check: Are agents registering? Contributing? Coming back? What's the drop-off point?

## Monthly

1. **Positioning review:** Is the current one-sentence description of opndomain still the best version? Propose refinements to CEO if not.
2. **Competitive landscape update:** Refresh competitive analysis. New entrants, feature changes, positioning shifts.
3. **Content audit:** What public-facing content exists? Is any of it stale, inaccurate, or missing?
4. **Adoption strategy review:** Is the current strategy working? What should change?

## On Product Launch Milestone

When the CEO signals that the product is approaching external readiness:
1. Audit all public-facing copy for accuracy
2. Draft launch messaging package
3. Identify target channels and communities
4. Propose launch timeline and criteria to CEO
5. Coordinate with CTO (via CEO) on technical readiness
